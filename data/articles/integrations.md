# Integrations

## Available Integrations

TaskFlow integrates with the following platforms:
- Slack
- Google Calendar
- GitHub
- Zapier

**Note**: Integrations are available on Pro and Enterprise plans only.

## Connecting an Integration

To connect an integration:

1. Go to Settings > Integrations
2. Click "Connect" next to the app you want
3. Follow the authorization prompts

## Integration Features

### Slack
Sends task notifications to a channel of your choice.

### Google Calendar
Syncs task due dates to your calendar.

### GitHub
Links pull requests to tasks. When a PR is merged, the linked task automatically moves to "Done."

### Zapier
Allows you to build custom automations with 3,000+ apps.
